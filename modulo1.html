História dos computadores

Voltar
A palavra “computador” vem do verbo “computar” que, por sua vez, significa “calcular”. Sendo assim, podemos pensar que a criação de computadores começa na idade antiga, já que a relação de contar já intrigava os homens.

Uma das primeiras máquinas de computador foi o "ábaco" instrumento de origem chinesa, criado no século V a.C, que era uma espécie de calculadora que realizava operações algébricas. No século XVII, foi criado pelo matemático escocês John Napier, o primeiro instrumento analógico de contagem capaz de efetuar cálculos logaritmos. Em 1640, o matemático francês Pascal inventou a primeira máquina de calcular automática. A primeira calculadora de bolso capaz de efetuar os quatro principais cálculos matemáticos, foi criada por Gottfried Wilhelm Leibniz. Já no século XIX, o matemático inglês Charles Babbage, conhecido como "Pai da Informática" criou uma máquina analítica que, a grosso modo, é comparada com o computador atual com memória e programas. Assim, a evolução dos computadores cresceu muito e foi dividida em quatro gerações. A 1ºgeração (1951-1959) os computadores funcionavam por meio de circuitos e válvulas eletrônicas como o Harvard Mark I, Colossus e ENIAC. A 2ºgeração (1959-1965) os computadores funcionavam por meio de transitores. Na 3ºgeração (1965-1970) os computadores funcionavam por circuitos integrados, foi nesse período que os chips foram criados e a utilização de computadores pessoais começou. Na 4ºgeração (1971 - até os dias atuais), os computadores diminuem de tamanho, aumentam a velocidade e capacidade de processamento de dados. São incluídos os microprocessadores com gasto cada vez menor de energia. Nesse período, há uma grande expansão dos computadores pessoais. Além disso, surgem os softwares integrados e a partir da virada do milênio, começam a surgir os computadores de mão. Ou seja, os smartphones, iPod, iPad e tablets, que incluem conexão móvel com navegação na web. E a 5ºgeração, onde estudiosos já preferem acrescentar que já estamos, é resumida em supercomputadores, inteligências artificais, criptomoedas etc.

O que são Clients?

Voltar

São usuários ou dispositivos que acessam e interagem com serviços ou recursos em uma rede. Os dispositivos de computação são desktops, laptops, smartphones, tablets ou aplicativos que, através de protocolos de rede, como a Internet, se conectam a servidores para fazer uma solicitação de serviço ou recurso e aguarda a resposta.

O que são Servers?

Voltar

Servidor é um recurso dentro de um sistema computacional, capaz de processar aplicações, armazenar dados e prestar serviços. Os servidores atendem a solicitações de clientes, como o fornecimento de páginas web, armazenamento de dados ou execução de aplicativos.

Como é uma estrutua básica de um HTML?

Voltar

Para a estrutura básica de uma página HTML, é necessária principais tags para que o documento seja corretamente interpretado pelos browsers.

ONDE:

DOCTYPE Sempre deve ser a primeira a aparecer em uma página HTML para indicar ao browser qual versão da linguagem usada

HTML Dentro desta tag que delimita o documento, ou seja, todas as demais tags da página devem estar nesse espaço.

HEAD Define o cabeçalho do documento.

META Específica qual conjunto de caracteres (character set ou charset) será usado para renderizar o texto da página. O UTF-8 contém todos os caracteres dos padrões Unicode e ASCII, sendo, portanto, o mais utilizado em páginas web.

TITLE Define o título da página, onde aparece na janela/aba do navegador.

BODY Marca o espaço no qual deve estar contido o conteúdo visual da página.

Entendendo a diferença de uma lista ordenada e não ordenada

Voltar

LISTA ORDENADA São usadas para indicar alguma sequência ou numeração. As tags usadas para criar uma lista ordenada são ol e li. A tag ol é usada para definir a lista e a tag li é usada para cada item da lista.

LISTA NÃO ORDENADA São usadas para listar itens, sem se preocupar com sua sequência. As tags usadas para criar uma lista não ordenada são ul e li. A tag ul é usada para definir a lista e a tag li é usada para cada item da lista.
